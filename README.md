## 调查概况

- **调查对象**：广东海洋大学在校本科生
- **样本量**：108份有效问卷
- **调查方法**：线上问卷调查（腾讯问卷平台）
- **问卷维度**：认知程度、认同态度、实践行为、教育评价

## 关键发现

1. **背诵率高但理解不足**：72.2%能完整背诵24字，但仅11.1%能详细解释内涵
2. **知行脱节明显**：86.1%认为核心价值观重要，但仅18.5%经常以之作为行为准则
3. **社会实践需求强烈**：76.9%希望通过社会实践接受教育，58.3%认为增加实践最有效
4. **诚信最受关注**：74.1%认为"诚信"对其影响最深，为12个词之首
